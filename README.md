# werehouse-database

This is a script file of a werehouse database.
Objects include:
- store
- work
- worker
- IO(inputs & outputs)
- manager
- general manager
- invoice
- driver
- truck
- product

# Useing the databse

1. create a new database with the exact name (db_werehouse)
2. click on the file.
3. click F5 on the keyboard to execute the script and DONE!
